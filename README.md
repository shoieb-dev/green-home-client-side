# 🏠 GREEN HOME PROPERTIES - Frontend

A modern and responsive frontend for a property booking platform built with React.js. Users can explore apartments, book properties, and interact with the system through a clean and intuitive UI.

---

## 🚀 Live Demo
🔗 [Live Site](https://niche-website-ms.web.app/)  
🔗 [Backend API](https://github.com/shoieb-dev/green-home-server-side)

---

## 📌 Key Features

This platform provides a seamless experience for both property seekers and administrators, featuring a fully responsive design for mobile and desktop.

### 🏠 For Users
* **Browse & Book:** Explore apartments with detailed information and book properties through a user-friendly interface.
* **Manage Bookings:** View your booked apartments and maintain full control with the ability to delete entries from your personal list.
* **Community Feedback:** Submit reviews for your stays and read feedback from others directly on the homepage.
* **Personal Dashboard:** Access a dedicated user dashboard to track all your activities and bookings.

### 🛠️ For Admins
* **Inventory Management:** Effortlessly add new apartment listings or remove existing ones from the platform.
* **Administrative Control:** Manage the team by adding new administrators to the system.
* **Admin Dashboard:** Utilize a specialized dashboard designed for high-level property and user management.

---

## ⚙️ Tech Stack

- React.js  
- JavaScript  
- Tailwind CSS  
- Axios API

## 🧩 Key Functionalities

- Dynamic routing and page rendering  
- API integration with backend services

- Responsive UI components
